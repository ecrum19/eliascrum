# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-03-02

### Added

- Vite-based build and development workflow for the Vue 3 website.
- Structured talks/posters data generation and PDF pipeline (`build:slides`).
- GitHub Pages deployment workflow with SPA fallback support.
- Weekly Google Scholar citation refresh automation.
- Expanded publications and talks interfaces with filtering, tags, and richer metadata.

### Changed

- Migrated from Vue CLI/webpack to Vite.
- Consolidated talks/posters source inputs into hidden compressed archives (`.talk-assets/*.tar.gz`).
- Updated repository documentation and release metadata.
